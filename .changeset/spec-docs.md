New `@ir-kit/spec-docs` package — render any supported spec format as standalone HTML via Scalar API Reference. Delegates HTML emission to `@scalar/core/libs/html-rendering` (the same renderer the official Express/Hono/Fastify integrations use). New `ir docs <spec> [--theme] --out docs.html` command.
